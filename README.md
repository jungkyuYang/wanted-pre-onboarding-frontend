# [원티드 프리온보딩 프론트엔드 - 선발 과제](https://github.com/walking-sunset/selection-task)

## 📦 실행방법

1. 프로젝트 디렉토리에서 다음 명령을 실행하여 종속성을 설치합니다.

   ```
   npm install
   ```

2. 다음 명령을 실행하여 애플리케이션을 시작합니다.

   ```
   npm start
   ```

애플리케이션은 기본적으로 http://localhost:3000 에서 실행됩니다.
